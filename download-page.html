macOS 菜单栏工具

今日印钞

把今天赚了多少钱挂在菜单栏里。支持月薪、时薪、工作日规则、节假日同步、加班时段和多种状态栏显示样式。

下载最新版 查看源码

核心功能

  • 菜单栏实时显示今日已赚
  • 按月薪或时薪计算
  • 支持午休、加班、节假日、补班
  • 支持自动保存和本地缓存

适合谁

  • 想实时看到今天赚了多少钱的人
  • 有明确上下班、加班规则的人
  • 想把收入数据做成菜单栏 HUD 的人

下载安装

推荐通过 GitHub Releases 下载打包好的 `.app` 或 `.zip`，解压后直接打开。

开源与反馈

欢迎提 Issue、提建议、提 PR。后续会继续完善 UI、节假日同步和自动更新。
