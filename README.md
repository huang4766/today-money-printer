# 今日印钞

一个原生 macOS 状态栏小应用，用来实时显示当天已经赚到的钱。

## 一句话介绍

把今天赚了多少钱挂在菜单栏里，边上班边看数字往上跳。

## 适合谁

- 想把工资、时薪、接单收入可视化的人
- 想在菜单栏里实时看到 `今日已赚` 的人
- 想按工作日、节假日、补班、加班规则来估算收入的人

## 功能

- 状态栏单行紧凑显示 `¥今日已赚 (+每秒增长)`
- 点击状态栏可展开配置面板
- 支持按月薪或按时薪计算
- 支持配置上下班时间、午休开始时间和午休时长
- 支持 `自动 / 开工 / 摸鱼 / 收工` 状态切换
- 支持扣除周末、录入法定节假日和补班工作日
- 支持启动时自动同步当年官方节假日，平时优先使用本地缓存
- 支持单独配置加班时段和加班倍率
- 配置自动保存在 `UserDefaults`

## 当前版本

- 产品名：`今日印钞`
- 平台：`macOS 14+`
- 形态：菜单栏常驻小工具
- 数据存储：本地配置，自动保存

## 下载

- 可以直接从 GitHub Releases 下载打包好的 `.app` 或压缩包
- 如果你自己构建，也可以按下面步骤本地生成 `今日印钞.app`

## 构建

```bash
chmod +x ./build_app.sh
./build_app.sh
```

构建完成后会在当前目录生成 `今日印钞.app`。

如果你本机执行时报 `swift` / SDK 版本不匹配，先把当前开发者目录切到完整 Xcode，而不是 `CommandLineTools`：

```bash
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
```

然后再重新执行 `./build_app.sh`。

## 运行

```bash
open ./今日印钞.app
```
