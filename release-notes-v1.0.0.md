# 今日印钞 v1.0.0

首个公开版本。

## 新增

- 菜单栏实时显示今日已赚金额
- 支持多种状态栏显示样式
- 支持按月薪或按时薪计算
- 支持配置上下班时间、午休开始时间和午休时长
- 支持 `自动 / 开工 / 摸鱼 / 收工` 状态切换
- 支持周末、法定节假日、补班工作日规则
- 支持加班时段和加班倍率
- 支持官方节假日同步与本地缓存
- 支持自动保存配置

## 使用方式

1. 下载 `今日印钞.app` 或对应压缩包
2. 打开后在菜单栏中找到 `今日印钞`
3. 点击菜单栏图标进入设置面板
4. 填写你的月薪、时薪、工时和规则

## 已知说明

- 当前主要面向 `macOS 14+`
- 如果遇到 Gatekeeper 拦截，需要在系统设置中允许打开
- 后续版本会继续完善 UI、自动同步和发布流程
